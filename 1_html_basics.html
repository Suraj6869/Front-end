HTML Basics

HTML stands for HyperText Markup Language. It is the standard language used to create web pages. HTML uses a system of tags to structure content on the web. Each tag has a specific purpose and can be used to define elements such as headings, paragraphs, links, images, and more.

Here are some basic HTML tags:

  • <h1> to <h6>: Heading tags, where <h1> is the largest and <h6> is the smallest.
  • <p>: Paragraph tag, used to define a block of text.
  • <a>: Anchor tag, used to create hyperlinks.
  • <img>: Image tag, used to embed images in a web page.
  • <ul> and <ol>: Unordered and ordered list tags, respectively.
  • <div>: Division tag, used to group elements together for styling or layout purposes.
